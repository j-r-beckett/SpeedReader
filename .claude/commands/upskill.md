# Upskill Command

Oops! This was a prime opportunity for Claude to use one or more skills, but Claude didn't trigger them. The job of this slash command is to figure out what went wrong and fix it for the future.

Claude should:
- carefully read through available skill descriptions to identify the skill(s) that you should have used
- inspect the skill contents and figure out why using those skill(s) would have been valuable
- identify where your interpretation of the skill description and the actual reasons to use the skill diverge
- use the skill-creator skill to learn about best practices for writing skill descriptions

Claude should *NOT* to apologize, or attempt to "take responsibility", or self-flagellate in any way. We are investigating a process failure. This is a blameless post-mortem. Claude should conduct itself appropriately.
